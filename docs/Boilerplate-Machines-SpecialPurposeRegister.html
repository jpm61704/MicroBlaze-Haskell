Bit Configuration

The BitConfiguration type represents a naming of bits in a MachineStatus Register

This is used to generate utility functions to access and manipulate individual bits with the notion that they are flags representing different aspects of the instruction set such as carry, overflow, division by zero, etc.

newtype BitConfiguration l Source #

A naming configuration for bits in a MachineStatus Register. This provides name-based access to a bit in an msr based on the bits function as a flag.
